🔍 کد هگزادسیمال (Hex Color) چیست؟

کد هگزادسیمال یا Hex Code یک روش استاندارد برای نمایش رنگ‌ها در طراحی وب و گرافیک است. این کد از ترکیب ۶ کاراکتر (حروف و اعداد) تشکیل شده که با علامت # شروع می‌شود. دو کاراکتر اول میزان قرمز (Red)، دو کاراکتر دوم میزان سبز (Green) و دو کاراکتر آخر میزان آبی (Blue) را مشخص می‌کنند.

🎯 کاربردهای کد هگز

  • طراحی رابط کاربری (UI) و تجربه کاربری (UX)
  • طراحی گرافیک و لوگو
  • برنامه‌نویسی وب با CSS و HTML
  • توسعه اپلیکیشن‌های موبایل
  • ویرایش تصاویر و ویدئو

چگونه از این ابزار استفاده کنم؟

روی جعبه رنگ کلیک کنید، رنگ مورد نظر را انتخاب کنید. سپس کد هگز به‌صورت خودکار به‌روز می‌شود. با کلیک روی دکمه «کپی کردن کد»، آن را در کلیپ‌بورد خود ذخیره کنید.

آیا کد هگز با RGB تفاوت دارد؟

بله، کد هگز یک نمایش ۱۶-تایی (Hexadecimal) از مقادیر RGB است. برای مثال رنگ قرمز در هگز #FF0000 و در RGB معادل rgb(255, 0, 0) است.

آیا این ابزار رایگان است؟

بله، این ابزار کاملاً رایگان و بدون نیاز به ثبت‌نام است. می‌توانید از آن در هر پروژه‌ای استفاده کنید.
